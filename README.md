## Struktur Repository

- `kode/`  
  Berisi seluruh source code/program untuk tiap tugas praktikum.
- `Laporan_Praktikum_Otomata_1.md`  
  Laporan untuk Praktikum Otomata 1.
- `Laporan_Praktikum_Otomata_2.md`  
  Laporan untuk Praktikum Otomata 2.
- `Visual/`  
  Dokumentasi gambar/screenshot pendukung laporan.

## Akses Kode Program

Untuk melihat dan menjalankan kode, masuk ke folder:

```bash
cd kode
```

Contoh program untuk Praktikum 2 (FSM Visual GUI):

```bash
python fsm_visual_gui.py
```

Jika environment Windows menggunakan launcher Python:

```bash
py fsm_visual_gui.py
```

## Akses Laporan Tiap Soal

- Laporan soal/praktikum 1: buka file `Laporan_Praktikum_Otomata_1.md`
- Laporan soal/praktikum 2: buka file `Laporan_Praktikum_Otomata_2.md`

Setiap laporan menjelaskan:
- deskripsi tugas,
- rancangan/konsep automata yang digunakan,
- implementasi program,
- hasil pengujian dan kesimpulan.
